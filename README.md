# ATM-Simulation-SimpleProject1

## Overview
This project simulates an ATM machine using Core Java. The system allows the user to authenticate with a PIN and perform basic banking operations.

## Features
✔ User Authentication  
✔ Check Account Balance  
✔ Deposit Money  
✔ Withdraw Money with Validation  
✔ Exit Option  
✔ Loop-based menu to continue operations  

## Technologies Used
- Core Java
- OOPS Concepts (Encapsulation)
- Scanner Class
- Switch-Case Control Flow

## How It Works
1. User enters a PIN.
2. If the PIN is correct, the menu is displayed.
3. The user can perform banking operations until they choose to exit.
